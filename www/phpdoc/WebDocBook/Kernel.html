WebDocBook\Kernel

class Kernel

Class Kernel

This is the central kernel of WebDocBook, fully static.

It only throws "classic" \Exception as it is used by Composer's commands scripts (when the namespace is not yet loaded).

The $_defaults table defines default required paths and variables.
