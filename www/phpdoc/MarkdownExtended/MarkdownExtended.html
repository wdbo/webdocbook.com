Class

MarkdownExtended\MarkdownExtended

class MarkdownExtended implements KernelInterface

PHP Markdown Extended Mother Class

This is the global MarkdownExtended class and process. It contains mostly static methods that can be called from anywhere writing something like:

MarkdownExtended::my_method();
